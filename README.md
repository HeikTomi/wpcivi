# CiviCRM Wordpress extended image

### Contains:

* Wordpress with:
  * CiviCRM
  * CiviCRM Admin Utils
  * Mosaics
  * Members
  * WP CLI
  * CV CLI
  * Civix CLI

### Notice

* Check ENVIRONMENT variables in docker-compose -file for your setup
* **Updating master branch will trigger new :latest docker image build**
